# Milz2 Return Support Site

米国から日本への帰国サポートサイトの静的HTML一式です。

## ページ構成

- `index.html` トップ
- `jobs.html` 求人を探す
- `moving.html` 引越し・手続き
- `insurance.html` 保険
- `housing.html` 住まい
- `articles.html` お役立ち記事
- `company.html` 法人の方へ
- `login.html` ログイン
- `signup.html` 無料会員登録
- `css/style.css` 共通CSS

## GitHub Pagesで公開する場合

1. このフォルダの中身をGitHubリポジトリへアップロード
2. GitHubの `Settings` → `Pages`
3. `Deploy from a branch`
4. Branchを `main`、Folderを `/root` に設定
5. `Save`

## ローカル確認

`index.html` をブラウザで開いて確認できます。
